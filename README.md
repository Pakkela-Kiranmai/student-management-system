# Student Management System

## Description

This is a Django-based web application that allows users to manage student records with CRUD operations.

## Features

* Add Student
* View Student List
* Update Student Details
* Delete Student
* Search Functionality

## Technologies Used

* Python
* Django
* HTML, CSS (Bootstrap)
* SQLite

## How to Run

1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run migrations: python manage.py migrate
4. Start server: python manage.py runserver
